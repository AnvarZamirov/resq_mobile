# ResQ Mobile Application

📱 Мобильное приложение для персональной безопасности ResQ

## Описание

ResQ Mobile - это мобильное приложение-компаньон для устройства персональной безопасности ResQ, предоставляющее пользователям интуитивную настройку, управление экстренными контактами и функции безопасности в реальном времени.

## 🎨 Философия UI/UX

- **Минималистичный интерфейс** – максимум 3 тапа до активации экстренного режима
- **Высококонтрастный дизайн** для ситуаций с низкой освещенностью
- **Тактильная обратная связь** для всех критических действий
- **Голосовое сопровождение** для доступности
- **Без логина** для экстренного режима

## 📲 Основные экраны

### 1. Emergency Screen (Главный экран)
- Большая красная кнопка SOS с удержанием 3 секунды
- Индикаторы статуса (GPS, Микрофон, Сеть, Батарея)
- Быстрые действия (Поделиться геопозицией, Чек-ин, Безопасная поездка)

### 2. Contacts Management
- Управление сетью доверия
- Добавление/редактирование экстренных контактов
- Статусы контактов и история уведомлений

### 3. Device Dashboard
- Информация об устройстве ResQ
- Управление устройством
- Статус подключения и батареи

### 4. Safety Timeline
- Журнал всех событий безопасности
- Чек-ины и экстренные активации
- Фильтры по типу событий

### 5. Emergency Overlay
- Полноэкранный режим во время активации SOS
- Таймер и статус отправки
- Индикация записи аудио

### 6. Onboarding
- Быстрая настройка за 60 секунд
- Запрос разрешений
- Добавление первого контакта

## 🚀 Ключевые функции

- **Умные режимы экстренной ситуации**
  - FOLLOW_ME (пользователь движется)
  - SILENT_ALERT (обнаружены громкие звуки)
  - HOME_EMERGENCY (гео-ограждение)
  - STANDARD (стандартный протокол)

- **Проактивные функции безопасности**
  - Планирование безопасного маршрута
  - Автоматические чек-ины
  - Дискретные режимы оповещения

- **Функции после экстренной ситуации**
  - Отчет о действиях
  - Сохранение в журнал
  - Экспорт для полиции

## 🛠 Технический стек

- **Framework**: Flutter (iOS/Android)
- **State Management**: Provider
- **Navigation**: GoRouter
- **Location**: Geolocator
- **Permissions**: Permission Handler
- **Storage**: SharedPreferences
- **Audio**: Record

## 🚀 Запуск проекта

1. Установите зависимости:
```bash
flutter pub get
```

2. Запустите приложение:
```bash
flutter run
```

## 🔒 Безопасность и приватность

- Сквозное шифрование всех коммуникаций
- Локальная обработка аудио (отправляются только метаданные)
- Автоматическая очистка данных через 72 часа
- Отсутствие отслеживания в неэкстренном режиме
